Introduction

This sample shows how to create the game, Color Copy - a game based off of Simon Says. The object of the game is to copy a sequence of commands (in this case, colors) in the same order to advance in difficulty. 

Some of the more complicated APIs used in this code include the following:

  1. Storyboard (for animations) - using DoubleAnimationUsingKeyFrames and LinearDoubleKeyFrame
  2. MediaElement (for sounds)

The full source code attached is everything that was submitted and published to the Windows 8 App Marketplace (02JAN2013).

Please note that I also left in the icons and logos that I used to publish to Windows 8. Those were included so that you may easily run the code without having to worry about finding proper images before testing the code. Those included images should be changed when making modifications to this code.

Building the Sample

There are no extra files or downloads required in order to build the program and play the game.

Step 1: Download the attached source code.

Step 2: Open Color Copy in Visual Studios 2012

Step 3: Run

Description

The code was made to be extensible. There are comments included in code where needed.


Colors are given arbitrary numeric values in the MainPage.xaml.cs; It does not matter what the values are. That way, the colors can be changed at anytime in the code or xaml and the game will still run.
